**1.¿Qué tipo de componente he creado?**

El componente que he creado, se trata de un componente de conversión de euros a dólares estadounidenses. El valor de dicha moneda, va fluctuando conforme pasa el tiempo, por lo que actualmente, el valor de la moneda en el componente, es de 1.17$.

**2.¿Cómo funciona este componente?**

Este componente funciona de una forma  muy sencilla. Tan solo tenemos que introducir un importe(euros) en el campo de texto de la interfaz, y tras pulsar el botón de “conversión”, aparecerá un texto justo debajo con el importe convertido a dólares. 
Este componente contiene propiedades propias y eventos propios como “conversion()”, en el que una vez pulsemos el respectivo botón, converitrá el importe introducido a dólares, y animaciones propias como "FadeTransition" , en el que cuando convertimos el importe a dólares, esta nueva tasa aparecerá con una transición de "fundido".

**3.Incorporar el componente en nuestro proyecto.**

Para poder incorporar este componente a un proyecto de Intellij, habrá que hacer lo siguiente. Crearemos un nuevo proyecto JavaFX (Gradle) y configuraremos las dependencias necesarias en nuestro build.gradle.

**4.Bibliotecas y recursos utilizados en el componente.**

Para realizar el componente, han sido necesarios una serie de bibliotecas y recursos para poder comprobar su funcionamiento. La herramienta que he empleado para visualizar la interfaz, ha sido “Scenebuilder”.
Las bibliotecas empleadas en el proyecto han sido las siguientes:
- JavaFX: Para la interfaz gráfica.
- JUnit: Para pruebas unitarias.
- Gradle: Para gestión de dependencias y empaquetado.
- SceneBuilder: Para diseño de FXML.
- GitHub: Para documentación web.
